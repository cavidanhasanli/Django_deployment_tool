# Django deployment Tool #


### --- How to install and use for Ubuntu 18.04 ###

First you need to install python3 and after install django you can run the project following bellow steps
if you have a python3 follow this step
PS:This program must also be used root.
```
$ https://github.com/cavidanhasanli/Django_deployment_tool.git
$ cd Django_deployment_tool
$ chmod +x ./deploy.sh
$ ./deploy.sh usage  #command list
$ ./deploy.sh create #create database
$ ./deploy.sh deploy #deploy
$ ./deploy.sh status #deploy status
```
